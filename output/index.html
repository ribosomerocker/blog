"denotation"? What kind of name is that for a website?

The name of this website is called "denotation" for a reason. It's mainly got to do with denotational semantics, which to my understanding is linking code to it's canonical mathematical abstraction, which I absolutely love. Writing code in hopes of keeping it as denotational as possible allows us to reuse more code and for details to be clearer.
